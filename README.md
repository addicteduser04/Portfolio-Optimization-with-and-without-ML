# Portfolio Optimization with and without Machine Learning

A quantitative finance project comparing traditional portfolio optimization methods with machine learning–based approaches for asset allocation and portfolio construction.

The objective is to analyze whether machine learning models can improve portfolio performance, risk management, and allocation efficiency compared to classical optimization techniques.

---

# Project Overview

This project explores:

- Classical portfolio optimization
- Machine learning–enhanced portfolio strategies
- Risk-return analysis
- Portfolio performance evaluation
- Financial data analysis and visualization

Traditional optimization methods often rely entirely on historical statistics, while machine learning models attempt to improve predictions and adapt to changing market conditions.

The project compares these approaches using quantitative finance techniques and performance metrics.

# Installation

Clone the repository:

```bash
git clone "https://github.com/addicteduser04/Portfolio-Optimization-with-and-without-ML"
```

Move into the project directory:

```bash
cd Portfolio-Optimization-with-and-without-ML
```

Install dependencies:

```bash
pip install -r requirements.txt
```

# Usage

Run the main project files and experiments to:

3. Train machine learning models
